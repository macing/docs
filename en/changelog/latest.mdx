  # New Drag and Drop Shelf

  <Frame>

  </Frame>

  ### New Features

 - Added drag-and-drop shelf for easy dragging of screenshots to other apps  
 - Added option to save screenshots as webp  
 - Added window snapping between pinned image windows

  ### Improvements and Fixes

 - Selecting annotation targets does not steal tool focus  
 - Fixed issue where screenshots in Lyricsx & BongoCat appeared blank  
 - Fixed crash caused by adding stickers  
 - Fixed conflict between mosaic and Spotlight tools

<Update label="v1.1.0" description="2025/4/30">
  # 新增数字点备注

  <Frame>
    ![v1.1.0 Jp](/en/images/v1.1.0.jpg)
  </Frame>

  ### 新增功能

  - 新增全新的数字点备注，并增加指针数字点
  - 新增表情贴纸
  - 新增保存或拷贝后，在右下角显示拖拽缩略图
  - 新增按下 <kbd>Command</kbd> 和 <kbd>Option</kbd> 时，滚动滚轮可调节第二、三个滑动条
  - 新增钉图窗口直角&圆角设置（钉图-右键菜单）
  - 新增钉图一次性隐藏全部（钉图-右键菜单）
  - 新增显示上次钉图（顶部菜单栏）
  - 新增截图选区大小自定义
  - 新增按住 <kbd>Option</kbd> 并拖拽标注对象时复制标注

  ### 优化与修复

  - 修复微信图片预览窗口截图空白的问题
  - 修复截图 Cursor 窗口自动隐藏的问题
  - 修复全屏截图会将截图提示一起截图
  - 修复按住 <kbd>Command</kbd> 截图多个窗口无法显示工具的问题
  - 修复滚动截图会截入预览图片的问题
  - 交互&动画&UI 多项优化
</Update>
