icon: "pipette"
---

1Capture supports quick color picking. Press the screenshot shortcut to enter capture mode. A preview window appears next to the cursor. The first line of the preview window shows the screen coordinates, and the second line displays the color value at the current cursor position.
  
Currently, Hex, RGB, and HSL color formats are supported. Press <kbd>Option</kbd> to switch between color formats, and press <kbd>C</kbd> to copy the color value.
