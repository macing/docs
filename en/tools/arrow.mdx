The Arrow tool provides directional annotations, making it a highly practical feature. 1Capture’s Arrow tool offers various settings to meet different needs.

<Frame>
  ![Arrow Toolbar Jp](/en/images/arrow_toolbar.jpg)
</Frame>

**Line Arrow:** The connector line of line arrow supports straight, curved, and polyline styles.  

**Thick Arrow:** The thick arrow can be resized using the slider.
