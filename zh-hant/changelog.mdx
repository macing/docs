  # 新增支援 10 種語言

  <Frame>
    ![v1.3.0 Jp](/zh-hant/images/v1.3.0.jpg)
  </Frame>
  <Card title="1Capture 下载" icon="circle-arrow-down" horizontal href="https://apps.apple.com/app/id6742318274">
    
  </Card>
  ### 新增功能

  - 介面新增支援 10 種語言  
  - 幫助文件及更新日誌新增支援 3/10 種語言  
  - OCR 支援 25 種語言自動識別  
  - 支援設定 F1~F0 快捷鍵  
  - 矩形／橢圓形工具支援設定邊框線為虛線  
  - 新增複製工具  
  - 新增 MOS 反向滾動設定  
  - 新增拖曳標註物件四角進行旋轉功能  
  - 新增截取游標設定項

  ### 優化與修復

  - 修復水印工具超出螢幕的問題
</Update>

<Update label="V1.2.0" description="2025/5/30">
  # 新增拖放架

  <Frame>

<Update label="v1.1.0" description="2025/4/30">
  # 新增數字點備註

  <Frame>
    ![v1.1.0 Jp](/zh-hant/images/v1.1.0.jpg)
  </Frame>
  ### 新增功能

  - 新增全新的數字點註記，並增加指標數字點
  - 新增表情貼紙
  - 新增保存或複製後，在右下角顯示拖曳縮圖
  - 新增按住 <kbd>Command</kbd> 和 <kbd>Option</kbd> 時，滾動滑鼠滾輪可調整第二、三個滑桿
  - 新增釘圖視窗直角與圓角設定（釘圖－右鍵選單）
  - 新增一次性隱藏所有釘圖（釘圖－右鍵選單）
  - 新增顯示上次釘圖（頂部選單列）
  - 新增截圖區域大小自訂
  - 新增按住 <kbd>Option</kbd> 並拖曳標註物件時複製標註

  ### 优化與修復

  - 修復微信圖片預覽視窗截圖空白的問題
  - 修復截圖 Cursor 視窗自動隱藏的問題
  - 修復全螢幕截圖會將截圖提示一併擷取的問題
  - 修復按住 <kbd>Command</kbd> 截取多個視窗時無法顯示工具的問題
  - 修復滾動截圖會包含預覽圖片的問題
  - 交互、動畫及 UI 多項優化
</Update>
